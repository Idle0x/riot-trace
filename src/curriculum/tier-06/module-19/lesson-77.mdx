### EXECUTION REQUIREMENTS
This is a Boss Fight. You are building the exact logic that your code editor uses to check if your `{ }` and `( )` brackets match up correctly. This is one of the most famous algorithmic tests in computer science.

You are given a string containing only parentheses `()` and brackets `[]`. 
You must write the `isValid(s)` function to determine if they are closed in the exact correct order. 

**The Stack Strategy:**
1. Loop through every character in the string.
2. If the character is an **open** bracket ( `(` or `[` ), `push` it onto your array stack.
3. If the character is a **closed** bracket ( `)` or `]` ):
   * `pop` the top item off your stack.
   * Check if the popped item is the correct matching pair. (e.g., if you found a `]`, the popped item MUST be a `[`).
   * If it doesn't match, return `false` immediately. The string is broken.
4. After the loop finishes, if there is anything left in your stack, it means an open bracket was never closed. Return `false`.
5. If the stack is completely empty, return `true`. It is perfectly balanced.

Execute perfectly.
