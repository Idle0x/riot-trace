A **Stack** is a data structure that follows the **LIFO** principle: *Last In, First Out*. 

Think of it like a stack of plates in a cafeteria. You can only add a new plate to the very top, and you can only remove a plate from the very top. You cannot pull a plate from the middle without shattering the stack.

### Why not just use an Array?
We *do* use an array under the hood to store the data. However, standard arrays give developers too much power. You can use `.splice()` to delete data from the middle, or `.unshift()` to add data to the beginning. 

If you are building an "Undo History" feature, allowing another engineer to inject a random action into the middle of the history timeline will corrupt the entire application. 

By wrapping the array inside a `Class`, we **restrict access**. We only expose a `push` method and a `pop` method. This guarantees the data can only ever flow in a strict LIFO order.
