A **Queue** is the exact opposite of a Stack. It follows the **FIFO** principle: *First In, First Out*.

Think of a line at a coffee shop. The first person to enter the line is the first person to get served. 



### The Queue Methods
In computer science, the actions of a Queue have specific names:
* **Enqueue:** To join the back of the line. (In JS arrays, this is `.push()`).
* **Dequeue:** To leave the front of the line. (In JS arrays, this is `.shift()`).

### When to use a Queue
If a server receives 1,000 requests in a single second, it cannot process them all instantly. It places them in a Queue. The server then pulls them off the front of the line one by one, ensuring the oldest request gets handled first. This prevents the server from crashing and guarantees fairness.
