XP counts, node health scores, storage in terabytes, credit balances — your apps are full of numbers. This lesson makes you fluent in how JavaScript handles numerical data.

### Number Types
JavaScript has one `number` type for both integers and floats (decimals). 

```javascript
// Integers
42
1944        // your xandeum-pulse commit count
201         // nodes on the network

// Floats
429.47      // TB of network capacity
78.3        // average health score
```

### Mathematical Operators
The machine calculates exactly like a calculator, using specific symbols:
```javascript
10 + 5      // 15  — addition
10 - 5      // 5   — subtraction
10 * 5      // 50  — multiplication
10 / 5      // 2   — division
10 % 3      // 1   — modulo (returns the remainder after division)
```

### The `typeof` Operator
Sometimes you receive data from an API and you don't know what it is. The `typeof` operator asks the machine to identify the data. It always returns a string.

```javascript
typeof 42           // returns "number"
typeof "hello"      // returns "string"
```
