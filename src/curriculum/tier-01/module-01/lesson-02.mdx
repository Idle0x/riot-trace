### The Numbers
XP counts, node health scores, storage in terabytes—your applications are driven by numbers. JavaScript has one `number` type for both integers and floats.

```javascript
// Mathematical operators
10 + 5      // 15  — addition
10 - 5      // 5   — subtraction
10 * 5      // 50  — multiplication
10 / 5      // 2   — division
10 % 3      // 1   — modulo (remainder after division)
```

### The Anatomy of Types
You must be able to verify what kind of data you are holding before you manipulate it. The `typeof` operator tells you exactly what the machine sees.

```javascript
typeof 42           // "number"
typeof "hello"      // "string"
```
