Every monitoring dashboard, every transaction log, every error message in xandeum-pulse — all of it starts as a string. Before you build anything, you must be able to talk to the machine and have it talk back.

### The String
A string is text data. It must be wrapped in quotes. Three types of quotes exist in JavaScript and they are not interchangeable by accident — each has a purpose:

\`\`\`javascript
// Single quotes — basic string
'Hello Matrix'

// Double quotes — use when string contains an apostrophe
"riot's dashboard"

// Backticks (template literals) — use when embedding variables
\`Node count: \${nodeCount}\`
\`\`\`

### Anatomy Breakdown — Character by Character
Look at this exact command:
\`console.log("System Online");\`

* \`console\` = The object. Built into every JS environment.
* \`.\` = Dot notation. Accessing a method on the object.
* \`log\` = The method name. "log" means print to output.
* \`(\` = Opening parenthesis. Starts the argument you are passing in.
* \`"\` = Opening double quote. Begins the string.
* \`System Online\` = The string content.
* \`"\` = Closing double quote. Ends the string.
* \`)\` = Closing parenthesis.
* \`;\` = Semicolon. Ends the statement.
