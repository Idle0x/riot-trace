### The String
Every monitoring dashboard and transaction log starts as a **String**. In JavaScript, strings are text data wrapped in quotes.

```javascript
'Single quotes'
"Double quotes"
`Backticks (Template Literals)`
```

### The Anatomy of a Log
```javascript
console.log("System Online");
// ^-----^ Object and Method
//         ^-----------^ The String Argument
```
