### The Context
Every monitoring dashboard, transaction log, and error message starts as a string. Before you can build *riot-sop* or query *xandeum-pulse*, you must be able to talk to the machine and have it talk back.

### The Anatomy
A string is text data. It must be wrapped in quotes. JavaScript provides three types of quotes, and they are not interchangeable by accident:

```javascript
'Single quotes' // Used for basic text
"Double quotes" // Used when text contains an apostrophe
`Backticks`     // Used for embedding variables (Template Literals)
```

**The Logging Mechanism:**
```javascript
console.log("System Online");
//  ^------^  = the console object
//          ^ = dot notation accessing a method
//           ^--^ = log — the method name
//               ^-----------^ = the string argument
```

### The Crucible
You must pass three consecutive tasks to clear this lesson. The environment will automatically advance you as you clear each stage.

* **Task 1 (Replication):** Output the exact boot sequence required by the system.
* **Task 2 (Mutation):** The provided code is broken because a single quote is colliding with an apostrophe. Fix it.
* **Task 3 (Synthesis):** Output two separate system messages back-to-back.
