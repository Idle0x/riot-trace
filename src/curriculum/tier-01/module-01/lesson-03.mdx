### The Boolean
The machine only understands binary states: On or Off. True or False. In JavaScript, these are called **Booleans**. We generate them using comparison operators.

```javascript
10 > 5    // true
10 < 5    // false
10 === 10 // true (Strict Equality - always use 3 equals signs)
10 !== 5  // true (Strict Inequality)
```

### Control Flow (If / Else)
We use Booleans to route the execution of our program. If a condition is true, open one gate. If it's false, open another.

```javascript
let gasPrice = 45;

if (gasPrice > 50) {
  console.log("Delaying transaction.");
} else {
  console.log("Executing transaction.");
}
```
