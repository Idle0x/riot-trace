### Encapsulation
Writing code line-by-line is fine for scripts, but software is built with reusable blocks. A **Function** is a machine. You pass raw materials in (parameters), the machine processes them, and it spits a finished product out (return).

```javascript
// 1. Defining the Function
function calculateTax(amount) {
  let tax = amount * 0.20;
  return tax; // <--- The 'return' keyword is critical. It hands data back.
}

// 2. Calling the Function
let finalTax = calculateTax(500); 
console.log(finalTax); // Logs 100
```

### The Crucible
* **Task 1 (Replication):** The system function is defined, but it hasn't been executed. Call it.
* **Task 2 (Mutation):** A financial calculation is running, but the data is trapped inside the function. Use `return` to extract it.
* **Task 3 (Synthesis):** Build a mathematical function from scratch. Define it, pass parameters into it, and log the returned data.
