### Ordered Lists
An array is a list of values held in a specific order. 
* **ZERO-INDEXED:** The first item in an array is at index `0`, not `1`.

```javascript
const nodes = ["nodeA", "nodeB", "nodeC"];
//             index 0   index 1   index 2

// Accessing elements via Bracket Notation
nodes[0]           // "nodeA"
nodes[1]           // "nodeB"
nodes[nodes.length - 1]  // "nodeC" (This is how you always get the last item)
```

### Mutating the Array
You can change the contents of an array even if it was declared with `const`. (Remember: `const` locks the *label*, not the internal data).

```javascript
nodes.push("nodeD");      // Adds to the END
nodes.unshift("nodeZ");   // Adds to the BEGINNING

nodes.pop();              // Removes the LAST item
nodes.shift();            // Removes the FIRST item
```

### Checking Contents
```javascript
nodes.indexOf("nodeB")    // Returns 1 (the index)
nodes.includes("nodeB")   // Returns true (boolean)
```
