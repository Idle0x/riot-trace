You've seen `const { data, error } = await supabase.query()`. You've seen `const [count, setCount] = useState(0)`. This syntax appears everywhere in modern JavaScript, especially in React.

### Object Destructuring
Instead of accessing properties one by one, you can "unpack" them directly into standalone variables.

```javascript
const node = { id: "abc", health: 95, network: "mainnet" };

// Old way (Verbose):
const id = node.id;
const health = node.health;

// Destructuring (Clean):
const { id, health } = node;
```
*Rule: The variable names inside the `{}` MUST perfectly match the keys in the object.*

### Array Destructuring
Arrays don't have named keys, so they are unpacked strictly by **position**.
```javascript
const coordinates = [45.12, -12.89, 100];

// Unpack by position
const [lat, lng] = coordinates; 
// lat = 45.12, lng = -12.89
```
