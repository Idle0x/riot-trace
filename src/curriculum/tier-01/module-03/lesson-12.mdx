### EXECUTION REQUIREMENTS
This is a Boss Fight. The terminal is yours. 

Given the starting `node` object, execute the following exactly:

1. Using a **ternary operator**, declare a `const healthStatus` that is `"HEALTHY"` if `node.health` > 80, else `"AT RISK"`. Log `healthStatus`.
2. Using an **`if` statement and logical operators (`&&`)**, log `"CRITICAL ALERT"` **only** if `node.health` < 50 AND `node.network` is `"mainnet"`.
3. Write a **switch** on `node.version` — `case "1.2.1"` logs `"CURRENT"`, `default` logs `"OUTDATED"`.

*(Note: We are using a basic object here, which we will formally learn in Module 4. Access its properties using dot notation, e.g., `node.health`)*.
