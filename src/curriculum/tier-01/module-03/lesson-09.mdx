Is the node healthy or offline? Should we show the warning banner? Do we have permission? Every piece of dynamic behavior in your apps runs through an `if` statement at some level.

### The Syntax

**1. Basic `if`**
Executes the block of code inside the `{}` only if the condition inside the `()` evaluates to `true`.
```javascript
const health = 95;

if (health > 80) {
  console.log("Node is healthy");
}
```

**2. `if-else`**
Provides a fallback path. One of the two blocks will *always* execute.
```javascript
if (health > 80) {
  console.log("Node is healthy");
} else {
  console.log("Node needs attention");
}
```

**3. `if-else if-else`**
Handles multiple conditions. **Important:** Conditions are evaluated top-to-bottom. The *first* true condition wins, and the rest are entirely skipped.
```javascript
if (health >= 90) {
  console.log("Excellent");
} else if (health >= 70) {
  console.log("Good");
} else {
  console.log("Critical");
}
```
