### EXECUTION REQUIREMENTS
I have seeded two tables:
1. `nodes` (`id`, `name`, `network`)
2. `telemetry` (`id`, `latency`, `node_id`)

Write a single SQL query that executes the following pipeline:
1. `SELECT` all columns `*`.
2. Start `FROM` the `nodes` table.
3. `JOIN` the `telemetry` table `ON` the proper primary/foreign key relationship.
4. Filter the results `WHERE` the `nodes.network` is `'mainnet'`.
5. Sort the results (`ORDER BY`) the `telemetry.latency` in ascending order (`ASC` - lowest latency first).
6. Restrict the output (`LIMIT`) to exactly `1` row.
