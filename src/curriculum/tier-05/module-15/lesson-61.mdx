You know how to get specific data. Now you need to order it. 
If a user wants to see their 10 most recent transactions, you don't ask the database for their entire 5-year history. You ask the database to sort it by date and stop after 10 rows.

### Sorting Data (ORDER BY)
Just like `array.sort()`, SQL allows you to order data based on a specific column. It goes *after* the `WHERE` clause (if you have one).

* `ASC` = Ascending (Smallest to Largest / A to Z). This is the default if you don't specify.
* `DESC` = Descending (Largest to Smallest / Z to A).

```sql
SELECT * FROM transactions ORDER BY amount DESC;
```

### Restricting Output (LIMIT)
If you only want the top 5 results, you use `LIMIT`. This is always the absolute last command in your query.

```sql
-- "Get the 3 largest transactions"
SELECT * FROM transactions ORDER BY amount DESC LIMIT 3;
```
