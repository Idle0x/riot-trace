### EXECUTION REQUIREMENTS
This is a Boss Fight. The terminal is yours. You are running a database cleanup script containing multiple commands. 

**Table:** `users`
**Columns:** `username`, `balance`, `role`

You must write exactly two SQL statements. 
**CRITICAL:** Because you are running multiple statements in one script, you **must** place a semicolon `;` at the end of each statement so the database engine knows where one stops and the next begins.

1. Write an `UPDATE` query to change the `balance` to `5000` for the user `WHERE` the `username` is `'Cipher'`.
2. Write a `DELETE` query to remove any rows `WHERE` the `role` is `'banned'`.

Execute the mutation perfectly.
