### THE FINAL GATES
You have completed the React Mental Model. You understand how JSX transpiles to objects, how State triggers renders, how Props pass data down, and how `useEffect` isolates side effects.

This is the Tier 03 Capstone. This is the exact pattern used to fetch and display data in every modern dashboard on earth.

### EXECUTION REQUIREMENTS
You are building the final `Dashboard` UI. I have provided an array of `mockData` outside the component. 

1. Initialize a piece of state called `nodes`, starting with an empty array `[]`.
2. Write a `useEffect` hook. Give it an empty dependency array so it only runs once.
3. Inside the effect, use your state setter function to load the `mockData` into your `nodes` state.
4. Down in the JSX return block, use `.map()` to iterate over your `nodes` state. 
5. For each node, return a `<div>`. Remember the `key` prop using the node's `id`! Render the node's `name` inside the div.

*Validation Protocol:* Upon build, the system will verify your state initializers, your effect dependencies, and scan the physical DOM to ensure both Node Alpha and Node Beta rendered perfectly.
