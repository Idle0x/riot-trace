In Tier 2, you learned how to use `.map()` to transform an array of objects into an array of strings or numbers. 

In React, we use `.map()` to transform an array of raw data into an array of **JSX Elements**. 



### The Dynamic List
You will rarely hardcode `<li>Node 1</li>`, `<li>Node 2</li>`. Instead, you will receive an array from your database and map it directly to the glass.

Because `.map()` returns a new array, and React knows exactly how to render arrays of JSX, we can write it directly inside our component's return block using curly braces `{}`.

```jsx
const users = ["Alice", "Bob", "Charlie"];

function UserList() {
  return (
    <div>
      <h2>Active Users</h2>
      {/* We open JS mode with { } and immediately map */}
      {users.map((user) => {
        // We MUST return JSX for every item!
        return <div className="user-card">{user}</div>;
      })}
    </div>
  );
}
```

### The Concise Map
Just like standard arrow functions, if you are only returning one element, you can drop the `return` keyword and the curly braces around the arrow function body.

```jsx
// Exact same logic, much cleaner syntax:
{users.map(user => <div className="user-card">{user}</div>)}
```
