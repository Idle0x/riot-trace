### EXECUTION REQUIREMENTS
This is a Boss Fight. You are combining Components, JSX, Variables, Events, and State.

1. Define a React component called `NodeControl`.
2. Inside it, initialize a piece of state called `power` starting at `0`.
3. The component must return a `<div>`.
4. Inside the `<div>`, place an `<h1>` that says exactly `Power: {power}`.
5. Below the `<h1>`, place a `<button>` element.
6. When the button is clicked, it must increase the `power` state by exactly `10`.
7. Call the `render()` function to mount your `NodeControl` component to the DOM.

*Validation Protocol:* When you hit build, the system will verify the initial text says `Power: 0`. It will then forcefully click your button twice at lightspeed. If the glass does not update to say `Power: 20`, you fail.
