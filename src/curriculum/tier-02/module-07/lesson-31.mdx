### EXECUTION REQUIREMENTS
This is a Boss Fight. The terminal is yours. 

You need to produce a leaderboard of IDs for healthy nodes. 
**CRITICAL:** You are not allowed to mutate the original `nodes` array. Remember that `.sort()` mutates.

1. Create a variable called `leaderboard`.
2. Use the spread operator `[...nodes]` to create a safe copy of the array.
3. Chain `.sort()` onto that copy to sort the nodes by health in **descending** order (highest first).
4. Chain `.filter()` to keep only nodes with `health` greater than 50.
5. Chain `.map()` to extract just the `id` strings.
6. `console.log()` the final `leaderboard` array.

If you chain this correctly, you will process complex data in a single, highly readable block of professional code.
