Calculate total XP. Sum all transaction amounts. Count nodes by network. Any time you collapse an entire array into a **single value** (a number, a string, or an object), that is `.reduce()`.



### The Anatomy of Reduce
`.reduce()` is the hardest array method for beginners to grasp because it tracks a "running total" as it loops.

```javascript
const numbers = [10, 20, 30];

// We need TWO parameters inside the inner function:
// 1. acc (Accumulator): The running total.
// 2. num (Current Item): The item we are currently looking at.

const total = numbers.reduce((acc, num) => {
  return acc + num;
}, 0); // <-- The '0' is the starting value of the accumulator!

console.log(total); // Logs: 60
```

### Tracing the Execution
If you don't understand how `reduce` works under the hood, read this trace carefully based on the code above:

* **Start:** `acc` is `0` (because we set it at the end).
* **Loop 1:** `acc` is `0`. `num` is `10`. We return `0 + 10`. 
* **Loop 2:** `acc` is now `10`. `num` is `20`. We return `10 + 20`.
* **Loop 3:** `acc` is now `30`. `num` is `30`. We return `30 + 30`.
* **End:** The array is finished. The final value (`60`) is saved to the `total` variable.
