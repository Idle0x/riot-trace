While `for...of` is great, modern JavaScript leans heavily into "functional" programming. Arrays have built-in tools (methods) attached to them for iterating. 

The most basic one is `.forEach()`.

### How It Works
Instead of writing a manual loop, you call `.forEach()` on an array, and you hand it a **function**. It will run that function one time for every single item in the array.

```javascript
const networks = ["mainnet", "devnet"];

networks.forEach((network) => {
  console.log("Deploying to: " + network);
});
```

### Arrow Functions (A Brief Intro)
Notice the `(network) => { ... }` syntax above. That is called an **Arrow Function**. It is just a shorter, modern way to write a function. 
1. Inside the `()`, you define the variable name that will hold the current item (e.g., `network`). You can name this anything you want.
2. The `=>` (arrow) points to the block of code `{}` you want to execute for that item.

```javascript
const data = [10, 20, 30];

// The word "num" represents the current item in the loop
data.forEach((num) => {
  console.log(num * 2); 
});
```
