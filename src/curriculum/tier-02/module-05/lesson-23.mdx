### EXECUTION REQUIREMENTS
This is a Boss Fight. The terminal is yours. There is no theory pane. 

You are going to chain your knowledge together to build a data pipeline. Do not mutate the original array. 

Given the `nodes` array:
1. Use `.filter()` to isolate **only** the `"mainnet"` nodes. Store this in a new variable.
2. Use `.map()` on that new filtered array to extract **only** the `health` values into an array of numbers.
3. Use `.reduce()` on that mapped array of numbers to calculate the **total sum** of the mainnet health scores.
4. `console.log()` the final sum.

*Note: As you get more advanced, you can "chain" these methods together (e.g., `array.filter().map().reduce()`), but for this boss fight, it is perfectly fine to do them one step at a time, saving each step to a new variable.*
