In *Xandeum Pulse*, you filter nodes by network (mainnet vs devnet), by health score, or by version. Every filter button you build triggers this exact pattern.



### Selective Collection
Like `.map()`, `.filter()` returns a **brand new array**. But instead of transforming the items, it decides whether to keep them or throw them away.

```javascript
const healthScores = [95, 45, 88, 62, 100];

const healthyOnly = healthScores.filter((score) => {
  return score >= 80;
});

console.log(healthyOnly); // Logs: [95, 88, 100]
```

### The Boolean Gate
The function you pass into `.filter()` acts as a gatekeeper. 
* If the function returns `true`, the item is kept in the new array.
* If the function returns `false`, the item is discarded.

```javascript
const users = [
  { name: "Alice", role: "admin" },
  { name: "Bob", role: "user" }
];

const admins = users.filter((user) => {
  // This evaluates to true or false
  return user.role === "admin"; 
});
```
