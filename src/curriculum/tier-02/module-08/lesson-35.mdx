### EXECUTION REQUIREMENTS
You are building the core pipeline for a monitoring dashboard. I have provided a mock database function `getNetworkData()` that returns a Promise resolving to an array of nodes.

1. Create an `async` function named `analyzeNetwork`.
2. Inside it, `await` the data from `getNetworkData()`.
3. Process the returned array: 
   * **Filter** it to keep *only* nodes where `status` is `'active'`.
   * **Sort** the active nodes by `health` in descending order (highest first).
4. `console.log()` the `id` of the node at index `0` (the healthiest active node).
5. Call your `analyzeNetwork()` function to execute the pipeline.
